hellhound.http.handlers

A collection of useful Ring handlers to be used with a Ring router. In case of HellHound, hellhound.http.route.

bad-request

(bad-request context)(bad-request {:keys [request]} msg)

default-handler

(default-handler req)

Given a request map it will return an empty responst map with status of 200.

not-found

(not-found req)

A handler which gets a request map and returns a ring response map with status of 404.

upgraded

(upgraded req)

A function which gets a request map and returns a response map with 101 status to upgrade the connection to websocket.
